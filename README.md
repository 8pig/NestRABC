## Description

Nestjs RABC Mysql/Mariadb 权限基础框架

## Installation

```bash
$ yarn install

docker-compse up -d
# 链接mysql 3306 
# 创建testdb表
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
